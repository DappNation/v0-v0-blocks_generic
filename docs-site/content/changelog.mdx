# Changelog

All notable changes to the ETHBLOX protocol and builder.

---

## [v0.3.0] - 2025-01-15

### Added
- Wallet-based build isolation (builds scoped to wallet address)
- "Switch Account" feature in wallet drawer
- Comprehensive API documentation
- Builder Weight display in wallet drawer
- Public gallery improvements

### Changed
- Updated to Next.js 15 and React 19
- Improved MetaMask connection flow
- Enhanced wallet drawer UI with stats and CTA

### Fixed
- Scroll issues on landing page
- Wallet drawer visibility and z-index
- Color contrast in dark mode

---

## [v0.2.0] - 2025-01-10

### Added
- Landing page with hero section
- Comprehensive footer with newsletter signup
- Documentation site structure
- Curve Guides (whitepaper) section
- Gallery page for public builds

### Changed
- Updated color palette to Ethereum-native aesthetic (greys, yellow, mint)
- Replaced logo with custom ETHBLOX icon
- Improved mobile responsiveness

### Fixed
- Build height limits removed (infinite stacking)
- Color theme persistence in localStorage

---

## [v0.1.0] - 2025-01-05

### Added
- Initial 3D voxel builder with Three.js
- Block placement, movement, and deletion
- Color palette with 3 themes (Default, Muted, Monochrome)
- Undo/redo functionality
- Save/load builds to Vercel KV (Redis)
- Keyboard shortcuts
- Auto-rotate camera mode
- Block dimension controls (width/depth sliders)

### Technical
- Next.js 15 App Router setup
- React Three Fiber integration
- Vercel KV for data persistence
- TypeScript throughout
- Tailwind CSS + shadcn/ui components

---
